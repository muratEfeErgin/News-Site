A responsive and modern landing page for a sports event, built to practice layout design using CSS Grid and Flexbox.

## Features

- **Sticky Navbar:** Responsive navigation with social media icons.
- **Hero Section:** Full-width background image with centered typography.
- **News Section:** Clean layout displaying the latest event updates.
- **Footer:** multi-column layout with a separate copyright section.

## Technologies Used

- HTML5 (Semantic Structure)
- CSS3 (Custom Variables, Media Queries)
- CSS Grid (Macro layout & page centering)
- Flexbox (Micro component alignment)
- FontAwesome (Icons)
